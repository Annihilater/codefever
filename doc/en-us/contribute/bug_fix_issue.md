# 提交问题

当在使用 CodeFever 过程遇到问题时, 可以通过在 Github 项目上提交问题来得到帮助。

Github 项目问题提交地址: [issues - PGYRE/codefever](https://github.com/PGYER/codefever/issues)

问题的类型可以是:

- `Bug` 修复
- 易用性改进
- 特性建议
- 文档或文案修改

在提交问题之前, 请确定以下工作已经完成:

- 问题可被修改
- 新特性符合基本逻辑
- 已搜索项目 `issue` 列表, 确定没有类似或相似问题
- 如果内容是 `Bug` 修复, 需要给出简单的测试用例
